# GasteMenos
Projeto da disciplina de Programação Orientada a Objetos (POO) da Universidade Federal do Cariri (UFCA).
GasteMenos é uma aplicativo de controle de despesas pessoais, feito para que qualquer um possa ter total controle de suas finanças, de maneira prática e acessível.
Ele permite que o usuário possa cadastrar uma categoria de movimentação, personalizando nome, tipo, limite e outras características. 
Ele também poderá realizar lançamentos de movimentação, sendo eles de despesa ou receita. Cada lançamento é feito por categoria. 
O aplicativo gera relatórios automaticamente, sendo completamente personalizável.
